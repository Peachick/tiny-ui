Use npm


## ðŸ”¨ Quick Start

```js
import { Button, Switch } from 'tiny-ui';

const App = () => (
  <>
    <Button btnType="primary">Click Me</Button>
    <Switch checked/>
  </>
);
```

And import style manually:

```js
import 'tiny-ui/dist/styles/index.css';
```
