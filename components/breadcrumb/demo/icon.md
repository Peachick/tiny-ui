### With Icon

Using an icon replaces a text or combination.

```jsx live
<Breadcrumb>
  <Breadcrumb.Item>
    <Icon name="home" />
  </Breadcrumb.Item>
  <Breadcrumb.Item>
    <a href="#/components/breadcrumb"><Icon name="inspection" />Project List</a>
  </Breadcrumb.Item>
  <Breadcrumb.Item>A Project</Breadcrumb.Item>
</Breadcrumb>
```
