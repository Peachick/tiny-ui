### Three sizes of Input

There are three sizes of an Input box: `lg`, `md` and `sm`.

```jsx live
<div style={{ width: 400 }}>
  <Input size="sm" placeholder="Small size" />
  <br/>
  <Input placeholder="Medium size" />
  <br/>
  <Input size="lg" placeholder="Large size" />
</div>
```
