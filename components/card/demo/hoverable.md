### Hoverable

Set `hoverable` to allow the card having a hover effect.

```jsx live
<Card hoverable title="Basic Card" extra={<a href="#">More</a>}>
  <Card.Content>
    <div>This is a content.</div>
    <div>This is a content.</div>
    <div>This is a content.</div>
  </Card.Content>
</Card>
```
